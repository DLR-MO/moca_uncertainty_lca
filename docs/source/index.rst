MOCA - Uncertainty in Life Cycle Assessment
===========================================

Welcome to the documentation for MOCA, the Python package for uncertainty analysis in Life Cycle Assessment (LCA)! This package provides robust tools for Monte Carlo simulation, and post-processing of LCA results, making uncertainty quantification in environmental modelling accessible and reproducible.

Key Features
------------

- Fast and parallelised Monte Carlo simulations for LCA
- Easy integration with existing LCA workflows
- Compatibility with Brightway2 databases and methods
- Post-processing and visualization tools
